## Training Content

### Module 1: Introduction to Cyber Risk Management
- **Duration**: 45 minutes
- **Difficulty**: Beginner
- **Topics**: Risk Assessment, NIST Framework, Risk Treatment
- **Sources**: NIST Cybersecurity Framework, ISO/IEC 27005:2022

### Module 2: Threat Analysis and Intelligence
- **Duration**: 60 minutes
- **Difficulty**: Intermediate
- **Topics**: Threat Intelligence, MITRE ATT&CK, APTs
- **Sources**: MITRE ATT&CK Framework, SANS Institute

### Module 3: Incident Response and Forensics
- **Duration**: 90 minutes
- **Difficulty**: Advanced
- **Topics**: IR Lifecycle, Digital Forensics, Evidence Collection
- **Sources**: NIST SP 800-61 Rev. 2, SANS IR Process

### Module 4: Secure Coding and Application Security
- **Duration**: 75 minutes
- **Difficulty**: Intermediate
- **Topics**: OWASP Top 10, Secure SDLC, Security Testing
- **Sources**: OWASP Top 10 2021, NIST SSDF

### Module 5: Cloud Security and Zero Trust Architecture
- **Duration**: 80 minutes
- **Difficulty**: Advanced
- **Topics**: Cloud Security, Zero Trust, Container Security
- **Sources**: NIST SP 800-207, Cloud Security Alliance

**Total Training Content**: 350 minutes (5.8 hours)
